# 常见问题手册（FAQ / 排错）

> 用户与运维向。开发/迭代问题见 [`DEVELOPMENT.md`](DEVELOPMENT.md)。

## 鉴权 / 登录

**Q：用了 2 小时就报 `Feishu API error (code=99991677): token expired`？**
飞书 `user_access_token` 本身只活 ~2 小时，靠 `refresh_token` 自动续期到 ~30 天。若没续期，多半是**授权时缺 `offline_access` scope**（没有它飞书不发 refresh_token）。
- 代码已改为**始终请求 `offline_access`**；但你需要**重新「飞书授权」一次**（旧 token 没带 refresh_token）。
- 后台「权限管理」确认 `offline_access` 已开通。重新授权后即可自动续期、不再 2h 掉线。

**Q：报「请先用飞书账号授权」/ 401 / 登录失效？**
顶部会出现「飞书登录已失效」横幅，点它回设置**重新授权**即可。助手只以**你本人身份**操作。

**Q：`403 / 无编辑权限`？**
该文档/表格不是你创建的、或没共享给你。用你本人有权限的账号操作，或在文档右上「分享」里把自己/应用加为可编辑协作者。助手**不会越权**（不用应用身份兜底）。

**Q：加密版打不开（解锁密码）？**
个人/加密版的 secret 是密文，需在 **设置 → 飞书鉴权 → 输入解锁密码** 解锁一次。密码在打包时由 `scripts/encrypt-secret.mjs` 设定（**整段复制，含可能的首尾符号**）；若你维护了多个加密包，注意每个包对应各自的解锁口令。

## 生成 / 导出

**Q：PPT / 看板 / 网站 点「导出 PDF」没反应？**
旧版因 sandbox 缺 `allow-modals` 导致 `window.print()` 被静默拦截，已修复。**重新加载扩展**后，用浮窗标题栏的 **🖨** 即可（PPT 会一页一张）。

**Q：提示说十几秒，实际要几十秒？**
PPT/建站/报告是多步 LLM 生成，**几十秒起**属正常（已把提示改为「约需几十秒」）。耐心等待或在面板点「取消」。

**Q：切到别的浏览器标签再回来，生成内容丢了？**
不会。AI 小程序/建站/幻灯片结果按页面缓存，回来自动恢复；保存过的在「我的网站/我的演示/看板药丸」里。

**Q：怎么调配色？**
浮窗标题栏 **🎨**：7 个品牌色 + 自定义取色器，实时给 PPT/网站/看板/图表换色；**↺** 恢复默认。

## 数据 / 升级

**Q：版本更新会丢失已保存的看板/PPT 吗？**
**不会。** 都存在 `chrome.storage.local`（PPT=`slides_decks_v1`，看板/网站=`dataviz_v1`），Chrome 更新扩展不清空。只有**卸载重装**或**扩展 ID 变了**（私钥 `extension-key.pem` 丢失/换了）才会丢。

**Q：知识库（Wiki）里的表格，浮窗/面板识别不出？**
已支持——会自动把 Wiki 解析成背后的真实表格/文档。识别不出时直接打开文档/表格本体即可。

**Q：会把数据发给大模型吗？**
只有执行你明确发起的任务时，才把**必要且有上限**的数据发给**你自己配置**的大模型；沙箱里运行的生成代码 `connect-src 'none'`，**拿了数据也发不出去**。

## 私有化

**Q：私有化某操作报 404，但功能应该支持？**
私有化飞书版本常落后于 SaaS。请求层会**自动回退** `/<服务>/vN/` → `v(N-1)`（见 [`PRIVATE_DEPLOYMENT.md`](PRIVATE_DEPLOYMENT.md) §6）。若仍 404，可能该端点在你的私有版未上线。

**Q：大模型请求被拒？**
私有化构建锁了 `VITE_OPENAI_ALLOWED_HOSTS`，Base URL 必须落在白名单内。

**Q：全屏「检查网络访问权限」？**
设了 `VITE_ALLOWED_CIDRS` 但浏览器（WebRTC mDNS）取不到内网 IP。改用网关层限制，或调整该配置。

## 安装

**Q：`.crx` 拖进去装不上？**
普通 Chrome 默认禁装商店外 `.crx`。个人用**解压 zip → 开发者模式 → 加载已解压**最稳；企业用 MDM 策略安装（见 [`enterprise/DEPLOY.md`](enterprise/DEPLOY.md)）。

**Q：大模型 Key 在哪填？**
每个用户在 **设置** 里自填（OpenAI 兼容，默认 DeepSeek），仅存本机、加密保存。

## 企业版（统一下发大模型）

**Q：公司发的版本，我没填 API Key 怎么也能用？**
企业版可由公司统一下发大模型配置：你用**本公司飞书账号授权**后自动获取，无需填 Key。设置里「大模型配置来源」可在「企业统一 / 手动」间切换（管理员可能锁定为仅企业统一）。

**Q：提示「你的账号不属于本企业，无法获取大模型配置」？**
你的飞书账号不在公司应用的可用范围/租户内。用**公司飞书账号**重新授权；仍不行找管理员把你加进可用范围。

**Q：公司换了大模型 key 后我这边报错？**
设置 →「大模型配置来源」→ 点 **重新获取**（清掉本地缓存的旧配置再拉一次）。

**Q：管理员怎么配？**
代理设 `LLM_BASE_URL/LLM_API_KEY/LLM_MODEL` + **必填 `FEISHU_TENANT_KEY`**，客户端构建 `VITE_LLM_FROM_PROXY=1`。详见 [`DEPLOYMENT.md` §3.4](DEPLOYMENT.md) 与 [`oauth-proxy/README.md` §5](oauth-proxy/README.md)。
